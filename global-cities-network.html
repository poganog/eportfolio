Global Cities Network

Electromagnetism Analogy

This project attempts to represent cities using an analogy to electromagnetism. We model two electric fields of opposite signals for each city:

Repulsive Field: $$R_i(x) = -\alpha \cdot \frac{Pop_i^\beta}{(d(x,x_i)+\epsilon)^k}$$

Attractive Field: $$A_i(x) = +\alpha^\prime \cdot \frac{Pop_i^{\beta^\prime}}{(d(x,x_i)+\epsilon)^{k^\prime}}$$

The resulting networks are formed by connecting each city to its strongest attractor.
